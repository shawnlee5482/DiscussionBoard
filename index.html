Discussionboard

Discussion Board Application for Sharing Thoughts

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Discussion Board

Discussion Board application enables user to discuss with other people for the specific topic.

User can create topics and add multiple posts for the topic and comment on the posts.

Add Topic

User can create a topic by using name, description and photos. For example, user can initiate discussion about a new laptop computer by uploading a camera image.

Post

User can add multiple posts to the topic. Other user and topic owner itself can add posts. Posts also can include images.

Image source

Thanks to FilePicker library, user can add images from multiple sources. For example, from local computer, take a photo now, Google drive, Facebook, etc..

User can vote for other person's posts

If user likes some post, user can add up vote. (down vote for the dislikes)

Statistics

User can check their statistics (number of topics, posts, comments)

Personal Profile

User can check personal profile
