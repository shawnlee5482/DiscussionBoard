Welcome to Discussion Board

Using Discussion Board, You can discuss specific topics with other people. Even you can add images from various sources including Camera, Google Drive, Facebook, etc..

Skill Set

MongoDB Express AngularJS Node.JS Bootstrap Filepicker JWT Auth Grunt Less Jasmine+Karma

Add Topic

You can create a topic by using name, description and photos. Topic list can be browsed in the Dashboard page. In the Dashboard page, you can see the summary of the topics (for example, number of posts and topic owner)

Add Post

You and other people can add posts to the topic. Posts also can include an image.

Image source

You can add images from multiple sources. For example, from local computer, take a photo now, Google drive, Facebook, etc.. Integrated FilePicker library

User can add comments and vote for posts

You can add comments by text only and vote for up/down

Statistics

You can see your or other's statistics details (number of topics, posts, comments)

Token Based Session Management

session is managed Using JWT. Password is stored encrypted in the server. Except login page and Signup page, access is protected by session management.

Unit testing

By using Jamine testing, unit test is performed for part of the source code. Karma runner and Jasmine is used together. Unit test is performed for part of the source code.

Grunt Build Tool

Grunt is used for build automation.
